Funções de Custo e Regularização

• Erro Quadrático Médio (MSE) para regressão

• Entropia Cruzada para classificação

• Regularização L1 e L2 para evitar overfitting

Tipos de Gradiente Descendente

• Batch Gradient Descent: usa todos os dados

• Stochastic Gradient Descent (SGD): uma amostra por vez

• Mini-Batch: compromisso entre velocidade e estabilidade

Overfitting e Underfitting

• Underfitting: modelo muito simples, alto viés

• Overfitting: modelo muito complexo, alta variância

• Equilíbrio entre complexidade e generalização

Under-fitting (Grau 1): A linha laranja é quase linear, não captura a oscilação dos dados.

(Grau 3): A curva acompanha bem o padrão geral dos pontos, sem exagerar.

Over-fitting (Grau 10): A curva apresenta oscilações extremas, tentando passar por quase todos os pontos — típico overfitting.

Batch Gradient Descent (azul): convergência suave e estável, mas mais lenta (usa todo o dataset a cada passo).

Stochastic Gradient Descent (SGD) (laranja): convergência mais rápida inicialmente, mas com alta oscilação (usa uma amostra por vez).

Mini-Batch Gradient Descent (verde): equilíbrio entre estabilidade e velocidade (usa pequenos lotes).

Equilíbrio entre Bias e Variância

O modelo ideal equilibra a complexidade para evitar tanto o underfitting quanto o overfitting, minimizando o erro total.
