Funções de Ativação

• Introduzem não-linearidade em redes neurais

• Sigmoide: comprime valores para [0,1]

• ReLU: simplifica cálculos e evita o problema de gradiente desvanecente

Sigmoide

σ(x) = 1/(1+e-x)

ReLU

f(x) = max(0, x)

Funções de Custo

• Quantificam o erro do modelo

• MSE: para problemas de regressão

• Entropia Cruzada: para problemas de classificação

Erro Quadrático Médio (MSE)

MSE = (1/n) Σ(yi - ŷi)2

Funções Convexas e Otimização

• Funções convexas garantem mínimo global único

• Facilitam a convergência de algoritmos de otimização

• Essenciais para modelos como regressão linear e logística
